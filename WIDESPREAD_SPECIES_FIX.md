# Widespread Species Background Sampling Fix

## The Problem

The script was originally designed for **regionally restricted native European ant species** (e.g., Camponotus species with ranges of 10-40°). The standard convex hull + buffer approach works well for these species.

However, for **widespread invasive/pantropical species** like Monomorium floricola, this approach fails catastrophically:

### Example: Monomorium floricola

**Geographic extent:**
- Latitude range: ~70° (spanning from temperate to tropical zones)
- Longitude range: ~180° (essentially global across tropics)

**What went wrong:**

1. **Occurrences (red dots)**: Tropical/subtropical regions worldwide
   - Central/South America
   - Tropical Asia
   - Southeast Asia/Pacific
   - Parts of Africa

2. **Background points (black dots)**: Generated by convex hull + buffer
   - Massive clusters in temperate North America
   - Massive clusters in temperate East Asia
   - **Completely inappropriate environmental space**

3. **Result**: The model tries to distinguish between:
   - ✓ Tropical environments (where species occurs)
   - ✗ Temperate environments (where convex hull generated background)

   This is trivially easy → **instant overfitting** → model failure

## The Solution

### Automatic Detection

The script now detects widespread species automatically:

```r
lat_range <- diff(range(p_coords$decimalLatitude))
lon_range <- diff(range(p_coords$decimalLongitude))

is_widespread <- (lat_range > 60 || lon_range > 100)
```

**Thresholds:**
- Latitude > 60° OR Longitude > 100° = widespread species
- This catches pantropical, invasive, and cosmopolitan species

### Environmental Similarity Sampling

For widespread species, instead of geographic buffering, we use **environmental similarity**:

**Step 1: Extract environmental values at occurrence points**
```r
presence_env <- terra::extract(current_predictors, p_coords)
```

**Step 2: Calculate environmental ranges (5th-95th percentile)**
```r
env_ranges <- apply(presence_env, 2, quantile, probs = c(0.05, 0.95))
```

**Step 3: Sample background globally on land**
```r
bg_candidates <- sf::st_sample(world, size = 50000)
```

**Step 4: Filter to environmentally similar regions**
```r
# Keep only points within environmental ranges of occurrence data
bg_filtered <- bg_candidates[within_environmental_ranges, ]
```

**Step 5: Randomly select 10,000 background points**
```r
bg_coords <- sample(bg_filtered, 10000)
```

### Fallback Mechanisms

**If environmental filtering is too strict (<100 points):**
- Relax to 1st-99th percentile
- Try again with wider environmental ranges

**If still insufficient:**
- Fall back to global land sampling
- Log warning for user review

## How This Fixes the Problem

### Before (Convex Hull):
```
Occurrences:  Tropical Asia, Central America, etc. (tropical climates)
Background:   North America, temperate Asia (temperate climates)
Model learns: "This species likes hot places vs cold places" → TRIVIAL → OVERFITTING
```

### After (Environmental Similarity):
```
Occurrences:  Tropical Asia, Central America, etc. (tropical climates)
Background:   Other tropical regions globally (similar tropical climates)
Model learns: "Within tropical environments, which specific conditions matter?" → REALISTIC
```

## Benefits

1. **No more inappropriate background points** - All background points are from environmentally similar regions
2. **Works for both native and invasive species** - Automatic detection switches strategy
3. **Prevents overfitting for widespread species** - Model has to work harder to distinguish presence from background
4. **More realistic niche modeling** - Background represents available habitat, not random global environments
5. **Maintains performance for regional species** - Convex hull still used for restricted ranges

## Technical Background

This approach is based on:

- **Barbet-Massin et al. (2012)** - Selecting pseudo-absences for species distribution models
- **VanDerWal et al. (2009)** - Environmental sampling vs geographic buffering
- **Elith et al. (2011)** - Environmental space in species distribution modeling

The key principle: **Background should represent the environmental space available to the species, not arbitrary geographic regions**.
