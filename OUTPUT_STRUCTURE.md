# SDM Output Folder Structure

## Overview

The updated script organizes outputs into dedicated subfolders for better file management.

## Folder Hierarchy

```
species/
└── {speciesKey}_{species_name}/
    └── SDM_maxnet/                      ← Main modeling folder
        ├── rasters/                     ← All .tif raster files
        ├── tables/                      ← All .xlsx data tables
        ├── response_curves/             ← Response curve plots
        ├── report_{speciesKey}/         ← Model report (generated by modelReport())
        └── *.png                        ← Main visualization plots
```

**Note**: All output folders (`rasters/`, `tables/`, `response_curves/`, `report_*/`) are within the `SDM_maxnet/` folder.

## File Locations

### 📊 Rasters Folder (`rasters/`)
All geospatial raster outputs (`.tif` files):
- `current_{speciesKey}.tif` - Current habitat suitability prediction
- `future_{speciesKey}.tif` - Future habitat suitability prediction
- `uncertainty_{speciesKey}.tif` - Model uncertainty (SD across CV folds)
- `change_{speciesKey}.tif` - Climate change impact (future - current)

### 📈 Tables Folder (`tables/`)
All data tables (`.xlsx` files):
- `initial_variable_importance_{speciesKey}.xlsx` - Variable importance before selection
- `correlated_variables_{speciesKey}.xlsx` - Pairs of correlated variables (>0.7)
- `selected_variables_{speciesKey}.xlsx` - Final selected variables with ranks
- `final_variable_importance_{speciesKey}.xlsx` - Variable importance after selection
- `optimization_results_{speciesKey}.xlsx` - Hyperparameter optimization results
- `jackknife_results_{speciesKey}.xlsx` - Jackknife test results
- `summary_{speciesKey}.xlsx` - **Master summary** with all model statistics

### 🖼️ Main Plots (SDM_maxnet root)
Visualization PNG files in the main `SDM_maxnet/` folder:

**Current predictions:**
- `current_{speciesKey}.png` - Global current suitability map
- `current_europe_{speciesKey}.png` - European current suitability map
- `data_inspection_{speciesKey}.png` - Occurrence and background points

**Future predictions:**
- `future_{speciesKey}.png` - Global future suitability map
- `future_europe_{speciesKey}.png` - European future suitability map

**Climate change:**
- `climate_change_impact_{speciesKey}.png` - Suitability change map

**Model diagnostics:**
- `uncertainty_{speciesKey}.png` - Prediction uncertainty map
- `roc_curve_{speciesKey}.png` - ROC curve
- `th_current_binary_maxTSS.png` - Current binary presence/absence (max TSS threshold)
- `th_future_binary_maxTSS.png` - Future binary presence/absence (max TSS threshold)

**Variable importance:**
- `initial_variable_importance_{speciesKey}.png` - Before variable selection
- `final_variable_importance_{speciesKey}.png` - After variable selection
- `jackknife_plot_{speciesKey}.png` - Jackknife test visualization

### 📁 Response Curves Folder (`response_curves/`)
Individual response curves for each environmental variable:
- `response_{variable_name}.png` - Standard response curve
- `marginal_response_{variable_name}.png` - Marginal response curve

### 📋 Model Report Folder (`report_{speciesKey}/`)
Comprehensive HTML report generated by `modelReport()`:
- `report.html` - Main report file
- Associated plots and data files

## Quick Reference

| File Type | Location | Count per Species |
|-----------|----------|-------------------|
| Rasters (.tif) | `rasters/` | 4 files |
| Tables (.xlsx) | `tables/` | 7 files |
| Main plots (.png) | `SDM_maxnet/` | ~13 files |
| Response curves (.png) | `response_curves/` | 2 × N variables |
| Model report | `report_{speciesKey}/` | 1 HTML + assets |

## Benefits of This Structure

✅ **Organization**: Rasters and tables in dedicated folders
✅ **Easy GIS workflow**: All `.tif` files in one place for batch loading
✅ **Data analysis**: All `.xlsx` files together for comparative analysis
✅ **Quick overview**: Main plots immediately visible in SDM_maxnet folder
✅ **Detailed exploration**: Response curves isolated in subfolder
✅ **Scalability**: Clean structure even with 100+ species

## Important Files

### Start with the Summary Table
**`tables/summary_{speciesKey}.xlsx`** contains:
- Species information and occurrence records
- Data sources breakdown
- Final variables used
- Model performance (AUC, TSS, sensitivity, specificity)
- Confusion matrix
- Feature class and regularization
- Climate change statistics

### For Analysis
- **Rasters**: Load all from `rasters/` for spatial analysis
- **Variables**: Check `tables/selected_variables_*.xlsx`
- **Performance**: Compare `tables/summary_*.xlsx` across species

### For Publication
- **Maps**: Use PNG files from main folder
- **Statistics**: Export from `tables/summary_*.xlsx`
- **Model details**: Reference `report_{speciesKey}/report.html`
